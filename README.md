# 🚀 Meu Repositório de Exercícios de HTML e CSS

## 👋 Introdução

Fala aí! Este é o meu **repositório pessoal** focado no aprendizado de **HTML5** e **CSS3**. O objetivo aqui é centralizar tudo o que estou praticando e documentar a minha evolução com essas linguagens fundamentais da web.

Aqui eu guardo desde as estruturas mais simples (como marcação de textos) até layouts mais elaborados com Flexbox e Grid. Se você está explorando este código, sinta-se à vontade para ver como estou organizando meus estudos e as soluções que encontrei pelo caminho!

---

## 🛠️ O que você vai precisar (Pré-requisitos)

Como este projeto utiliza **tecnologias nativas da web**, você não precisa baixar compiladores ou linguagens de programação pesadas no seu PC. 

### 1. Um Navegador Web
Qualquer navegador moderno (Google Chrome, Firefox, Edge ou Brave) serve para abrir e interpretar os arquivos deste repositório.

### 2. Um Editor de Texto (Recomendado)
Para visualizar o código "por baixo do capô", recomendo o **VS Code**. Ele facilita muito a leitura com cores e sugestões.

---

## 🏗️ Como baixar e preparar o projeto

Siga estes passos para colocar os exercícios para rodar na sua máquina:

### 1. Baixe o código
Se você não quiser usar o Git agora, pode baixar o código diretamente:
1. Clique no botão verde **"<> Code"** no topo desta página.
2. Selecione **"Download ZIP"**.
3. Extraia o arquivo em uma pasta de sua preferência.

---

### 📂 Passo 2: Como visualizar os exercícios

Existem duas formas simples de ver o resultado dos códigos:

#### Opção A: O jeito rápido (Direto no navegador)
1. Abra a pasta onde você extraiu o projeto.
2. Localize o arquivo chamado `index.html` de qualquer exercício.
3. **Clique duas vezes** no arquivo. Ele abrirá automaticamente no seu navegador padrão mostrando o resultado final.

#### Opção B: O jeito profissional (Com Live Server no VS Code)
Se você estiver com a pasta aberta no **VS Code**:
1. Vá na aba de Extensões (Ctrl+Shift+X) e instale a extensão **"Live Server"**.
2. Clique com o botão direito sobre o arquivo `index.html` do exercício desejado.
3. Selecione **"Open with Live Server"**.
4. **Vantagem:** Sempre que você alterar o CSS ou o HTML e salvar, o navegador atualizará sozinho!
